Listas

Lista no ordenada

Estructura básica 
    <ul>
        <li> Item 1 </li>
        <li> Item 2 </li>
        <li> Item 3 </li>
    </ul>
Ejemplo
  • Item 1
  • Item 2
  • Item 3

Lista ordenada

Estructura básica 
    <ol>
        <li> Item 1 </li>
        <li> Item 2 </li>
        <li> Item 3 </li>
    </ol>
Ejemplo
  1. Item 1
  2. Item 2
  3. Item 3

Lista de Definición

Estructura básica 
    <dl>
        <dt> Término 1    </dt>
        <dd> Definición 1 </dd>
        <dt> Término 2    </dt>
        <dd> Definición 2 </dd>
    </dl>
Ejemplo
Término 1
Definición 1
Término 2
Definición 2

Tablas

Estructura básica 
    <table>
        <caption> Título de la tabla </caption>
        <tr>  <!-- fila 1 -->
            <th> Table head </th>
            <th> Table head </th>
            <th> Table head </th>
        </tr>
        <tr>  <!-- fila 2 -->
            <td> Table data </td>
            <td> Table data </td>
            <td> Table data </td>
        </tr>
    </table>

Adicionalmente podemos usar las siguientes etiquetas para subdividir la tabla en tres secciones diferentes, con el fin de poder darles posteriormente formato a través de CSS:

<thead>
Etiqueta de cabecera de la tabla>
<tbody>
Etiqueta del cuerpo de la tabla
<tfoot>
Etiqueta del pie de tabla

Ejemplo

Título de la tabla
col 1 col 2 col 3
row 1 cell 11 cell 12 cell 13
row 2 cell 21 cell 22 cell 23
row3 cell 31 cell 32 cell 33
foot 1 foot 2 foot 3

Por último, podemos expandir las celdas colocando, dentro de las etiquetas <th> y <td> los siguientes atributos:

colspan="n"
Indica que la celda se expandirá n columnas hacia la derecha
rowspan="n"
Indica que la celda se expandirá n filas hacia abajo
